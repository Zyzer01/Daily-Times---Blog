# Daily Times - BLOG

A simple blog that allows you create, delete and read posts via route parameters

## Prerequisites

You should have nodejs installed of at least version 10 or higher

## Installation

Use the node package manager (npm) to install the node modules.

```bash
npm init
```
Install expressjs.

```bash
npm install express
```
Install ejs.

```bash
npm install ejs
```
Install body-parser.

```bash
npm install body-parser
```
Install mongoose

```bash
npm install mongoose
```

Compile TailwindCSS

```bash
npm run watch-css
```

## Run App

```bash
node app.js
```
